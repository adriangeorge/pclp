# String Methods

| `Metodă` | Descriere |
|---|---|
| `.upper()` | Convertește toate caracterele în majuscule. |
| `.lower()` | Convertește toate caracterele în minuscule. |
| `.title()` | Convertește prima literă a fiecărui cuvânt în majusculă. |
| `.find(sub)` | Caută `sub` și returnează indexul primei apariții. Returnează `-1` dacă nu este găsit. |
| `.replace(vechi, nou)` | Înlocuiește toate aparițiile textului `vechi` cu textul `nou`. |
| `.startswith(sub)` | Returnează `True` dacă string-ul începe cu `sub`, altfel `False`. |
| `.endswith(sub)` | Returnează `True` dacă string-ul se termină cu `sub`, altfel `False`. |
| `.split(separator)` | Împarte string-ul într-o listă de `sub`-stringuri, folosind `separator`. Dacă nu e specificat, împarte după spațiu. |
| `.isnumeric()` | Returnează `True` dacă toate caracterele sunt numerice, altfel `False`. |
| `.isalpha()` | Returnează `True` dacă toate caracterele sunt litere din alfabet, altfel `False`. |
| `.strip()` | Elimină spațiile albe de la începutul și sfârșitul string-ului. |
| `.rjust(width, fillchar)` | Aliniază string-ul la dreapta într-un câmp de lățime `width`, folosind `fillchar` pentru umplutură. |
