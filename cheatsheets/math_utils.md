# Math Module

| `Funcție / Constantă` | Descriere |
|---|---|
| `math.pi` | Constanta Pi (π≈3.14159...). |
| `math.e` | Constanta e (numărul lui Euler ≈2.71828...). |
| `math.sqrt(x)` | Calculează rădăcina pătrată a lui `x`. |
| `math.pow(x, y)` | Calculează `x` la puterea `y` (xy). Echivalent cu `x` ** y. |
| `math.sin(x)` | Calculează sinusul lui `x`. Atenție: `x` trebuie să fie în radiani. |
| `math.cos(x)` | Calculează cosinusul lui `x`. Atenție: `x` trebuie să fie în radiani. |
| `math.radians(grade)` | Convertește un unghi din grade în radiani. |
| `math.factorial(x)` | Calculează factorialul lui `x` (x!). |
| `math.floor(x)` | Rotunjește `x` la cel mai apropiat întreg inferior (rotunjire în jos). |
| `math.ceil(x)` | Rotunjește `x` la cel mai apropiat întreg superior (rotunjire în sus). |
| `math.exp(x)` | Calculează ex. |
| `math.log(x, base)` | Calculează logaritmul lui `x` în `base` dată. |
| `math.fabs(x)` | Returnează valoarea absolută (modulul) lui `x` ca float. |
